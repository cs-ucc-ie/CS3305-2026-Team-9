Built with Security in Mind

Your files and data are protected at every layer

🔐

AES-256-GCM Encryption

Client-side encryption using the Web Crypto API. Files are encrypted in your browser before they ever leave your device.

🧂

Salted Password Hashing

All passwords are hashed with unique salts using Werkzeug's security module. Plaintext passwords are never stored.

🛡️

CSRF Protection

Every form submission is verified with a unique CSRF token to prevent cross-site request forgery attacks.

⏳

Rate Limiting

Login attempts are rate-limited to prevent brute force attacks. Accounts are temporarily locked after too many failed tries.

🔑

Secure Token Generation

Share links use cryptographically secure random tokens, making them impossible to guess or enumerate.

📁

File Validation

Uploads are validated by file extension and MIME type with a strict whitelist. Maximum file size is enforced server-side.
